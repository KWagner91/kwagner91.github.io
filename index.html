Coding for our Future

"Why would I want to start to code?" I should convince you? The Internet is an essential part of our lives nowadays, just imagine the anarchy without it...

By learning to code you are part of the gang who shapes the Internet, create tools for other people and try to make all our lives easier (or make money)

Mental Benefits

To code or not to code, that is your question? Learning to code helps you understand how Website, Apps and Software is build.

Coding is all about problem solving and being creative, which can be quite challenging and frustrating. However, by overcoming your coding problems, you feel like a Coding Master of Disaster ;-)

If you code long enough, you will start dreaming in 0 and 1 as well... Just kidding, it hasn't happened to me yet! You only start to understand nerdy jokes like: "There are only 10 types of people"

Other Benefits

Coding can be done from home, from a library or wherever you find a computer. You don't even need a powerful machine!

If you code from home, you have the comfort of your apartment and pyjamas :-P
